# perfstat: Java Bean's for libjavaperfstat APIs 

To Use this java web project, add dependencies as follows,

```xml
    <dependency>
      <groupId>com.ibm.aix</groupId>
      <artifactId>perfstat</artifactId>
      <version>1.0.0</version>
    </dependency>
```
